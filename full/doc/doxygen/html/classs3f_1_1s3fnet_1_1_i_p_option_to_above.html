Detailed Description

Parameters for IP messages popping up.

IP will send parameters to the protocols above with a pointer to an IPOptionToAbove object. Note that emulation session is implemented as an intercept session in IP, which passes the entire IP packet. Since this data structure is not used by emulation session, we do not need to include fields, such as TOS, IDENT, and OFFSET.
