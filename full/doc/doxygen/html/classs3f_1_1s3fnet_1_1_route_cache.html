Detailed Description

Base class for any routing caches.

Stores a limited subset of RouteInfo values for faster lookup than in the Trie structure.

This class is designed to allow varied cache implementations regardless of the Trie implementation used at the cost of a single additional memory access vs a static, non-configurable cache as part of ForwardingTable's data.
