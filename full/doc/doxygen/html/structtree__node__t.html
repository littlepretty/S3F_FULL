Detailed Description

Struct representing a tree node.

A single node for use in a software combining tree. Forms one piece of the fast_tree_barrier_t.

int tree_node_t::sense

The node's local sense.

Functions much more like global sense, with local sense thread-specific. Used for releasing the barrier. It doubles as the release futex if release futexes are enabled, thus it is an integer vs a bool. Children are released by their parent's sense.

int tree_node_t::arrival_count

Current number of arrivals.

The number of nodes that have already arrived and have signaled this node (their parent) that they are waiting to be released. This is atomically incremented by each child node and doubles as the arrival_futex if arrival futexes are enabled, in which case it causes ARRIVAL_CHILDREN wakes instead of one at the proper time. This can still be preferrable to spinning, even with sched_yield.
