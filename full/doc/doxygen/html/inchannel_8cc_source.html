111  void InChannel::insert_process_pri( ProcessPri pp )
112 
113  Called to put a description of a Process to activate in the
114  InChannel's list of such. The ProcessPri structure identifies
115  the Process's owner (needed because the Process code body is
116  an Entity method), the Process code body, whether the association
117  persists (bound) after an activation, and the priority on the list.
118 
119  The method does a linear insertion into the list, placing the
120  element immediately before the first entry it encounters
121  (scanning by increasing priority) with a priority that is strictly
122  larger. This implies that insertions with the same priority are
123  ordered FCFS among themselves.
