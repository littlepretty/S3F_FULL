Detailed Description

Data structure to store coalesced TrieData sets.

This structure can be used to save space if there are duplicate RouteInfo entries. However, insertion time will go up due to checking for duplicates. Since this is effectively a map to the proper pointer, it introduces at least one additional memory access into the lookup-chain.
