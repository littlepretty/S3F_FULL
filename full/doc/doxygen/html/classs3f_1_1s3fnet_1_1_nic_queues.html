Detailed Description

A list of NIC queues.

Class NicQueues implements a static queue management module. It provides the most possible flexibility for designating a queue type in the dml script. The user can specify a queue type. If this type is registered in NicQueues, the creation function for it will be returned. Actually, the same method is applied elsewhere, e.g., dynamic protocol session creation.
