Detailed Description

class ProcessPri is a helper class used internally to order processes to be activated when an activation arrives at the InChannel.

It holds all that is needed to execute the activation. In addition it has a priority field used to order the list (Processes with lower priority value execute earlier) and a flag that indicates whether the process is "bound" to the InChannel, meaning that it remains on the list even after an activation. Unbound Processes are removed.
