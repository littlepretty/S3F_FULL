Detailed Description

The S3FNet simulation interface class.

The S3FNet network simulation interface class for interacting with the underlying S3F API.

Parameters
tlthe number of simulation threads to create (pthreads).
ltpsthe time scale of the clock ticks in terms of log base 10 clock ticks per second.
isOpenVZEmuwhether the OpenVZ-based emulation is supported. The true/false value is passed through DML (one of the top few lines).
openvz_emulation_typeindicates the type of openvz-based emulation, local (single physical machine) or distributed.
of_ctrl_typeindicates the type of simulated openflow controller. 0 means non-entity-based controller (default), interacting with simulated openflow switches through function calls; 1 means entity-based controller that aligned to a timeline like other S3F entity, and controller is a protocol session.
of_ctrl_modelindicates the model of simulated non-entity-based openflow controller. 0 means no controller 1 means OF_LEARNING_SWITCH_CONTROLLER refer openflow_interface.h for more configuration
