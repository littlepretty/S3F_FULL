void s3f::s3fnet::DmlObject::display ( int  indent = 0)
virtual

Print out the name of this DmlObject to the standard output.

This function should be overridden by the subclasses. For example, a host should print something like "Host 3". The parameter indent indicates the number of white spaces to be used before the name is printed out. This is so that we can print out the entire network with correct indentations. (Maybe we should print out the name to an arbitrary file stream. But it's not yet useful at this point).

Reimplemented in s3f::s3fnet::Net, s3f::s3fnet::Host, s3f::s3fnet::NetworkInterface, and s3f::s3fnet::Link.
