Detailed Description

A generic protocol message carrying a payload as a continuous byte stream or a list of data chunks.

There are two types of the payload this message is carrying. The payload data can be represented as a continuous byte stream or byte array. If this is true, the payload pointer should be of type byte* that points to a byte array or simply NULL. The latter case is when the data is intentionally not included in simulation for better efficiency. Also, the real_length should be the number of real bytes of this payload in the target system (rather than in the simulator). The data can also be a list of data chunks, in which case the list could have both fake data and real data blocks. This situation is identified by having real_length set to be zero. The real length of the payload can be obtained by invoking the DataChunk::totalRealBytes method.
