Detailed Description

Data structure to store uncoalesced TrieData sets.

Only useful to measure overhead associated with abstracting TrieData storage to an array. Otherwise performs no useful function over a Trie that stores data in its leaves. Also requires copying items to and from the array in addition to deleting source data to prevent losing it, thus adding both memory and computational overhead.

IMPORTANT: Cannot store TrieData items of varying size – all must return exactly the same value for triedata->size() since there will be implicit casting of raw memory.
